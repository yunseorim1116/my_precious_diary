## 개발 진행사항
```

- 그날 그날 느꼈던 감정을 기록해보세요. <post>
- 등록한 감정기록 조회가 가능합니다. <get>
- 감정기록은 당일에만 수정이 가능합니다. <put>
- 감정기록 삭제는 언제든지 가능합니다. <delete>
- 나의 감정기록에 회고 댓글을 달 수 있습니다. <delete>
- 회고 댓글 삭제가 가능합니다. <delete>

```
